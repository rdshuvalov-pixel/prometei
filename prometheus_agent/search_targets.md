# Цели обхода (Tier) — URL по одному на строку

Добавь свои карьерные страницы и борды. Строки, начинающиеся с `#`, игнорируются.
Маркер **`Tier:`** в строке задаёт tier для следующих URL до следующего `Tier:`.

Tier: demo
https://example.org

# Tier 1 — раскомментируй и подставь реальные URL
# https://careers.example.com/jobs

# Tier 4
# https://www.linkedin.com/jobs/search/...
